# Bluebird Server

### Application Stack

-   PHP
-   PHPUnit
-   Lumen
-   MySQL
-   Swagger

### How to run

1. 이 레포지토리 클론

2. 다른 팀원들에게 .env 파일 받아오기

3. 아래 명령어 실행해서 `vendor packages` 설치 및 `Swagger` 세팅

    ```
    $ composer install

    ```

4. 아래 명령어로 서버 실행
    ```
    $ php -S localhost:3000 -t public
    ```
